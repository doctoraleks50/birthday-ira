# Сценарій листівки

1. **Відкрити** — intro, без «казки»
2. **Привітання** — «З Днем Народження, кохана!» + текст → підпис «твій коханий» → кнопка **Далі**
3. **Букет піонів (3D)** — камера наближається з далека; кожна пелюстка окрема геометрія → кнопка **Взяти**
4. **Зоряне небо** — падаючі зірки + текст «Загадай бажання, поки падає зірка…»

Музика вимкнена.
